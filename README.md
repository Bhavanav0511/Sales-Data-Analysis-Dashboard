# Sales Data Analysis Dashboard

## Project Overview
This project analyzes SuperStore sales data using Microsoft Power BI. The dashboard helps businesses understand sales performance, profit trends, and regional performance.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- CSV Dataset

## Dashboard Features
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Interactive Slicers (Region, Category, Market)

## Dataset
SuperStore Orders Dataset

## Outcome
Built an interactive dashboard that provides insights into business performance and supports better decision-making.
